AI Madplan Generering

Konfigurer OpenAI til automatisk generering af madplaner.

API-nøglen er sikret server-side. Den håndteres via Supabase Edge Function og sendes aldrig til browseren.

Nøglen administreres via supabase secrets set OPENAI_API_KEY=sk-... i terminalen.

Minimum 16.000 for 7-dages madplaner. Højere = mere detaljeret, men dyrere.

Kun for GPT-5.1. Lavere = konsistent, højere = kreativ. Standard: 0.7
